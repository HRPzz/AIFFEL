## 내용 정리

- 지도 학습
  - 분류 : 데이터 x의 여러 feature 값들을 이용하여 해당 데이터의 **클래스 y를 추론** 하는 것
  - 회귀 : 데이터 x의 여러 feature 값들을 이용하여 연관된 다른 데이터 y의 **정확한 값을 추론** 하는 것
